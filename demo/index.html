What makes something an AI Agent?
Not just a chatbot — an agent has these four things:

System Prompt

Defines the agent's role, personality, and rules. This is what makes it YOUR agent, not just a generic chatbot.

Memory

The list of past messages. The agent reads the full history before every response — that's how it remembers context.

Tools

Functions the agent can call — web search, calculator, email, database. Tools are what let it act beyond just talking.

The Loop

The agent keeps running — observe, reason, act — until the goal is achieved or you stop it. That's what separates it from a one-shot response.

 Path 2: Python Agent (Google Colab)
~20 lines. Runs in your browser. No laptop setup required.

# ─── Step 1: Install Groq (run once) ────────────────────────────────────────
!pip install groq -q

# ─── Step 2: Import and connect ─────────────────────────────────────────────
from groq import Groq

client = Groq(api_key="YOUR_GROQ_API_KEY")   # paste your key from console.groq.com

# ─── Step 3: Define your agent's role ───────────────────────────────────────
# Change this system prompt to make it YOUR agent
SYSTEM_PROMPT = """
You are a helpful study assistant for students at Kabale University, Uganda.
You explain AI concepts in simple, clear language.
When asked about real-world applications, give examples from Uganda and Africa.
Keep answers concise and encouraging.
"""

# ─── Step 4: Start the conversation history (this IS the memory) ────────────
history = [{"role": "system", "content": SYSTEM_PROMPT}]

# ─── Step 5: The agent loop ──────────────────────────────────────────────────
print("Agent is ready! Type 'quit' to exit.\n")

while True:
    user_input = input("You: ")
    if user_input.lower() in ["quit", "exit"]:
        print("Agent: Goodbye! Keep building.")
        break

    # Add user message to memory
    history.append({"role": "user", "content": user_input})

    # Send full history to the model (this is how it "remembers")
    response = client.chat.completions.create(
        model="llama-3.3-70b-versatile",
        messages=history,
        max_tokens=512
    )

    reply = response.choices[0].message.content

    # Add agent reply to memory
    history.append({"role": "assistant", "content": reply})

    print(f"\nAgent: {reply}\n")

How to run it: Open Google Colab → New Notebook → paste the code → replace YOUR_GROQ_API_KEY → Run All. That's it.

What to build next
Once your first agent works, here are natural next steps:

Add a Tool

Give your agent the ability to search the web, do calculations, or read a file. In Python, a tool is just a function you define and pass to the model. In n8n, it's a node you connect.

Give it a Real Task

Change the system prompt to something in your own field. Test it with real questions from your discipline. See where it helps and where it fails — that's how you improve it.

Add Persistent Memory

Save the conversation history to a file or database so the agent remembers previous sessions. In n8n, use a Google Sheets or Supabase node. In Python, write history to a JSON file.

Deploy it

Share your agent with others. In n8n, activate the workflow and share the webhook URL. In Python, wrap it in a simple Flask or FastAPI app and host it free on Render or Railway.
